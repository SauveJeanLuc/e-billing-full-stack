
# Electricity-Billing

System is for Selling Electricity Online.




## Installation

1. Fork the project
2. Clone The Forked Repo
3. Run back-end-service with 
```bash
  cd back-end-service
  mvn spring-boot:run
```
4. Run front-end-service with npm
```bash
  cd front-end-service
  npm install
  
```
